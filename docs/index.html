Apa itu WARGAGO?

WARGAGO adalah solusi komprehensif untuk digitalisasi pengelolaan RT/RW yang mencakup:

  • Manajemen Data Warga - Sistem KYC otomatis dengan OCR dan face detection
  • Sistem Keuangan Terintegrasi - Transparansi penuh untuk iuran dan pengeluaran
  • Agenda Komunitas - Manajemen kegiatan dengan reminder otomatis
  • Polling Digital - Voting demokratis dengan hasil real-time
  • Marketplace Lokal - Platform jual-beli antar warga
  • Dashboard & Pelaporan - Visualisasi data lengkap berbasis role

Manajemen Warga

Menambah Warga Baru

1

Buka Menu Warga

Klik menu "Warga" di sidebar, lalu klik tombol "+ Tambah Warga Baru"

2

Upload KTP/KK

Upload foto KTP atau KK. Sistem akan otomatis ekstrak data menggunakan OCR.

3

Verifikasi Data

Periksa data hasil OCR dan lakukan koreksi jika diperlukan.

4

Face Detection (Opsional)

Ambil foto wajah untuk verifikasi identitas tambahan.

5

Simpan & Kirim Undangan

Simpan data dan sistem akan otomatis mengirim kode undangan via WhatsApp/SMS.

Admin dapat mengubah data warga kapan saja:

  • Informasi personal (nama, NIK, tanggal lahir, dll)
  • Alamat dan nomor rumah
  • Kontak (telepon, email)
  • Status keanggotaan (aktif, pindah, meninggal)

Perhatian!

Membuat Polling

Admin dapat membuat polling untuk keputusan bersama:

Jenis Polling

üìä Single Choice

Warga hanya bisa memilih satu opsi (Ya/Tidak, Pilihan A/B/C)

‚úÖ Multiple Choice

Warga bisa memilih beberapa opsi sekaligus

Langkah Membuat Polling

1. Menu Polling ‚Üí Buat Polling Baru
2. Isi judul dan deskripsi polling
3. Tambahkan opsi jawaban (min. 2 opsi)
4. Pilih jenis polling (single/multiple choice)
5. Tentukan mode voting (anonim/named)
6. Set deadline polling
7. Publikasikan

‚úÖ Warga akan menerima notifikasi push

Mengelola Agenda

Buat dan kelola kegiatan RT/RW dengan mudah:

Membuat Kegiatan Baru

  • Judul kegiatan: Nama acara (misal: Kerja Bakti, Pengajian, Rapat RT)
  • Tanggal & Waktu: Jadwal pelaksanaan
  • Lokasi: Tempat kegiatan
  • Deskripsi: Detail acara
  • RSVP: Aktifkan jika butuh konfirmasi kehadiran
  • Reminder: Set notifikasi otomatis (H-1, H-3, H-7)
üí°

Tips: Aktifkan RSVP untuk kegiatan yang memerlukan persiapan (makanan, tempat duduk, dll) agar bisa estimasi peserta.

Fitur untuk Warga

1. Dashboard

  • Agenda kegiatan mendatang
  • Polling aktif
  • Pengumuman terbaru

2. Profil & Keluarga

Kelola data pribadi dan keluarga:

  • Lihat dan edit data diri
  • Lihat anggota keluarga
  • Update kontak darurat
  • Ganti password

3. Iuran

üìä Lihat Tagihan

üí≥ Bayar Iuran

Sebagai Penjual:

  • Kelola pesanan masuk
  • Update status pengiriman
  • Lihat laporan penjualan

Catatan Marketplace

Marketplace WARGAGO adalah platform internal RT/RW. Transaksi dilakukan langsung antar warga. WARGAGO tidak memfasilitasi pembayaran atau pengiriman.
